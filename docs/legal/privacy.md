# Privacy Policy

Last updated: **17.02.2026**

This Privacy Policy explains how **NEKOSBEST** (“we”, “us”, “our”) processes information when you use the **nekos.best website**, the **NEKOSBEST API**, and related documentation.

**NEKOSBEST**  
Website: https://nekos.best  
Contact email: contact@nekos.best  

## 1. Scope

This Privacy Policy applies to:
- The **nekos.best website**
- The **NEKOSBEST API**
- Documentation and infrastructure required to operate these services

The services are intended for **developers** and are **not directed at individuals under the age of 13**.

## 2. Data We Process

We do not require user accounts and do not collect directly identifying personal data such as names or email addresses.

The following technical data may be processed automatically:

### 2.1 IP Address
- Used temporarily for:
  - Rate limiting
  - Abuse prevention
  - Security and DDoS mitigation
- IP addresses are not used to identify users beyond these technical purposes.

### 2.2 Logs
Technical logs may include:
- IP address
- Request headers
- Timestamps
- Error information

### 2.3 Request and Response Data
- API request and response payloads are processed only to fulfill requests
- Payloads are not permanently stored

### 2.4 Analytics
We use **Umami Analytics**, which is:
- Self-hosted
- IP-anonymized
- Cookie-free

Analytics data is used only to understand general usage patterns and improve service reliability.

### 2.5 Error Monitoring
We use **Sentry** to monitor errors and stability.  
Error reports may include technical information such as request metadata and IP-derived data, but are used solely for debugging and reliability purposes.

## 3. Purpose of Processing

Data is processed exclusively for:
- Operating the website, API, and documentation
- Rate limiting and abuse prevention
- Security and service stability
- Monitoring performance and detecting errors

## 4. Legal Basis

Where applicable, processing is based on **legitimate interest**, including:
- Protecting the service from abuse
- Ensuring availability and reliability
- Maintaining technical security

## 5. Infrastructure & Service Providers

The following providers are used to operate the service:

- **Hetzner** – hosting and infrastructure
- **Cloudflare** – Proxy, CDN and DDoS protection
- **Umami** – self-hosted, anonymized analytics
- **Sentry** – error monitoring and diagnostics
- **GitHub** – hosting of public documentation

Data is not shared with third parties beyond what is required to operate the service.

## 6. Data Retention

- **Cloudflare logs:** up to 7 days
- **Local/server logs:** up to 30 days
- Analytics and error data are stored only as long as necessary and, where applicable, in anonymized form

## 7. Data Sharing and Selling

- Personal data is **not sold**
- Data is **not shared** outside required infrastructure providers

## 8. User Rights

We do not maintain user accounts or persistent user profiles.  
Stored data generally cannot be linked to an individual beyond an IP address.

Where applicable, you may request:
- Information about processed data
- Deletion of retained technical data
- Objection to processing

Requests can be sent to: **contact@nekos.best**

## 9. Cookies

The nekos.best website does not use tracking cookies.  
Umami analytics operates without cookies.

## 10. Changes to This Policy

This Privacy Policy may be updated when necessary.

Relevant changes will be announced via our **Discord community**.  
The date at the top reflects the current version.

## 11. Contact

For privacy-related questions, contact:  
**contact@nekos.best**
